# 🤖 FinBot – Financial Advisory Chatbot Website

FinBot is a web-based financial advisory chatbot built as a **group project at IIT Patna**, designed to provide financial guidance in **native languages**.  
I contributed mainly to the **frontend development** using React.js, focusing on UI design and chatbot integration.

---

## 🚀 Live Demo

👉 Website: https://finbottt.netlify.app

## ✨ Features

- Financial chatbot interface  
- Supports native language interaction  
- Responsive UI  
- Clean React component structure  
- User-friendly chat layout
